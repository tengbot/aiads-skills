# How to Use AIADS Skills

AIADS Skills is a public kit for using AI agents to create ad creative assets.

Main site: https://aiads.im

## 1. Start with a product brief

Use `templates/ad-brief-template.md`.

A good brief includes:

- product name
- product category
- target customer
- pain point
- main benefit
- proof point
- offer
- target platform
- creative constraints

## 2. Generate hooks

Use `hook-skill` or `prompts/hooks.md`.

Ask for:

- pain point hooks
- curiosity hooks
- comparison hooks
- objection hooks
- demo hooks
- offer hooks

Choose 3 to 5 hooks for testing.

## 3. Generate scripts

Use `ad-script-skill` or `templates/ad-script-template.md`.

Generate:

- 15 second script
- 30 second script
- captions
- voiceover draft
- CTA variants
- scene notes

## 4. Generate storyboard

Use `templates/storyboard-template.md`.

Convert each script into:

- scene duration
- visual action
- product shot
- on-screen text
- voiceover
- transition
- video generation notes

## 5. Generate variations

Use `creative-variation-skill` or `templates/creative-variation-template.md`.

Create variations across:

- hook
- angle
- CTA
- opening scene
- caption style
- platform format
- voiceover tone

## 6. Localize ads

Use `ad-translation-skill` or `prompts/ad-translation.md`.

Localize:

- hooks
- captions
- CTAs
- voiceover
- platform-native wording

## 7. Build ad tool pages

Use `aiads-site-skill` and `templates/landing-page-template.md`.

Recommended P0 pages:

- `/ad-video-generator`
- `/product-image-to-video`
- `/url-to-video`
- `/ugc-video-ads`
- `/product-video-generator`
- `/ai-hook-generator`
- `/ai-ad-script-generator`
- `/ad-translation`
- `/pricing`

## Safety note

Do not invent proof, reviews, awards, or performance guarantees.

Do not promise guaranteed ROI, conversions, sales, or viral results.

Use safer language like draft creatives, faster creative testing, localized ad assets, and creative variations.
