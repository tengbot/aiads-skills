# AI Ad Page Launch Checklist

Use this checklist before publishing any AI ad tool page.

Main site: https://aiads.im

## Page basics

- Unique title
- Unique description
- One H1
- Self canonical
- Clear primary CTA
- Useful examples
- Related tools
- FAQ with real answers
- No empty sections
- No coming soon copy on indexable pages

## Required page sections

1. Hero
2. Tool or workflow entry
3. Input examples
4. Output examples
5. How it works
6. Use cases
7. Related tools
8. FAQ
9. Pricing or credits CTA

## AI ad page quality bar

The page should answer:

- What input does the user provide?
- What ad asset does the user get?
- Which platform is it for?
- Which audience is it for?
- What should the user test next?

## Safe claims

Use:

- draft ad creatives
- faster creative testing
- creative variations
- localized captions
- product video concepts
- UGC-style scripts

Avoid:

- guaranteed ROI
- guaranteed conversions
- guaranteed sales
- guaranteed viral results
- fake testimonials
- fake awards

## Sitemap rule

Only add the page to sitemap when it is complete.

Do not add:

- empty page
- unfinished page
- noindex page
- duplicate page
- query URL
- placeholder page

## Footer rule

Only link finished pages from the footer.

If a page is planned but not complete, keep it out of footer and sitemap.
