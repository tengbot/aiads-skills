# DTC Ads Prompts

Prompt patterns for direct-to-consumer ad creatives.

Main site: https://aiads.im

## DTC ad angle prompt

```text
Product: {{product}}
Category: {{category}}
Customer: {{audience}}
Pain point: {{pain_point}}
Main benefit: {{benefit}}
Differentiator: {{differentiator}}
Offer: {{offer}}
Platform: {{platform}}

Create 8 DTC ad angles.
For each angle, include:
- hook
- buyer tension
- product reveal
- proof moment
- CTA
- ideal visual opening
```

## Founder style ad prompt

```text
Create a founder-style short video ad.

Founder story: {{founder_story}}
Product: {{product}}
Why it exists: {{why}}
Audience: {{audience}}
Offer: {{offer}}

Return:
- 3 hooks
- 15 second script
- 30 second script
- scene-by-scene storyboard
- captions
- CTA variants
```

## Objection handling prompt

```text
Product: {{product}}
Objection: {{objection}}
Audience: {{audience}}
Proof: {{proof}}

Write 5 ad scripts that answer this objection without sounding defensive.
Each script should include hook, proof, product moment, CTA, and caption text.
```
