# UGC Video Ads Prompts

Prompt patterns for UGC-style ad drafts.

Main site: https://aiads.im

## UGC angle prompt

```text
Product: {{product}}
Audience: {{audience}}
Pain point: {{pain_point}}
Benefit: {{benefit}}
Platform: {{platform}}
Tone: {{tone}}

Create 8 UGC ad angles.
For each angle, include:
- angle name
- hook
- creator point of view
- 3 scene outline
- CTA
- notes
```

## UGC script prompt

```text
Product: {{product}}
Audience: {{audience}}
Benefit: {{benefit}}
Objection: {{objection}}
Offer: {{offer}}
Platform: {{platform}}

Write 3 UGC video ad scripts:
1. review style
2. problem solution style
3. comparison style

Each script should include hook, body, CTA, captions, and scene notes.
```
