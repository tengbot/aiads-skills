# TikTok Ads Prompts

Prompt patterns for TikTok-style short ad creatives.

Main site: https://aiads.im

## TikTok ad angle prompt

```text
Product: {{product}}
Audience: {{audience}}
Pain point: {{pain_point}}
Benefit: {{benefit}}
Offer: {{offer}}
Tone: {{tone}}

Create 10 TikTok ad angles.
Each angle should include:
- hook
- visual opening
- short script
- on-screen text
- CTA
- why it fits TikTok

Rules:
- write for mobile viewers
- keep the first 3 seconds sharp
- avoid fake proof
- avoid guaranteed results
```

## TikTok hook batch

```text
Generate 25 TikTok hooks for this product.

Product: {{product}}
Category: {{category}}
Audience: {{audience}}
Main promise: {{promise}}
Common objection: {{objection}}

Group hooks by:
- curiosity
- problem
- confession
- test result
- comparison
- offer

Return hooks under 12 words when possible.
```

## TikTok storyboard prompt

```text
Turn this ad idea into a 15 second TikTok storyboard.

Ad idea: {{ad_idea}}
Product: {{product}}
CTA: {{cta}}

Return:
- scene
- duration
- visual
- on-screen text
- voiceover
- editing note
```
