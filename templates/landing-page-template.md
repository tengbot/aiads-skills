# Ad Landing Page Template

Use this template for AI ad tool landing pages.

Main site: https://aiads.im

## Metadata

```yaml
path:
title:
description:
h1:
canonical:
primary_cta:
secondary_cta:
```

## Required sections

1. Hero
2. Tool or workflow entry
3. Input examples
4. Output examples
5. How it works
6. Use cases
7. Related tools
8. FAQ
9. Pricing or credits CTA

## Hero pattern

```text
H1: {{tool_keyword}}
Subtitle: Create {{output_type}} for {{audience}} using {{input_type}}.
CTA: Generate ad creatives
```

## Related tools pattern

Always link to nearby ad pages:

- AI Ad Video Generator
- Product Image to Video
- URL to Video
- UGC Video Ads
- Product Video Generator
- AI Hook Generator
- AI Ad Script Generator
- Ad Translation

## Quality bar

- One clear H1.
- Unique title and description.
- Self canonical.
- Real examples.
- Clear CTA.
- Useful FAQ.
- No empty sections.
- No coming soon copy on indexable pages.
