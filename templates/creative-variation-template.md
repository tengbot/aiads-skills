# Creative Variation Template

Use this template to create a batch of meaningfully different ad creative variations.

Main site: https://aiads.im

## Variation brief

```yaml
product:
category:
audience:
base_angle:
base_script:
platform:
offer:
number_of_variations: 10
constraints:
```

## Variation dimensions

Change one or more of these per variation:

- hook
- ad angle
- opening scene
- proof point
- objection
- CTA
- caption style
- voiceover tone
- platform format
- pacing

## Quality bar

A real variation should change the idea, not only the wording.

Weak variation:

```text
Buy now
Shop now
Get it now
```

Better variation:

```text
Pain point test
Founder story test
Product demo test
Before-after test
Objection handling test
Offer-driven test
```

## Rules

- Generate at least 10 variations.
- Avoid tiny wording changes disguised as tests.
- Do not invent proof or reviews.
- Do not promise guaranteed sales.
- Make each variation executable as a short ad video.
