# Storyboard Template

Use this template to turn an ad script into video generation notes.

Main site: https://aiads.im

## Brief

```yaml
product:
platform:
duration:
aspect_ratio:
ad_angle:
hook:
script:
visual_style:
brand_voice:
cta:
```

## Scene format

```yaml
scene:
  number:
  duration:
  goal:
  visual:
  product_shot:
  camera_motion:
  on_screen_text:
  voiceover:
  transition:
  generation_notes:
```

## 15 second default structure

```text
0-3s: Hook and pattern interrupt
3-6s: Pain point or desire
6-10s: Product reveal and demo
10-13s: Benefit moment
13-15s: CTA
```

## 30 second default structure

```text
0-3s: Hook
3-7s: Problem or desire
7-12s: Product reveal
12-18s: Demo
18-23s: Benefit moment
23-27s: Objection answer
27-30s: CTA
```

## Rules

- Make every scene visually clear.
- Keep product visible in key scenes.
- Use simple camera directions.
- Keep claims aligned with the supplied brief.
