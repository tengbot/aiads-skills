# Ad Script Skill

## Purpose

Use this Skill when an AI agent needs to create ad scripts for UGC video ads, product videos, paid social creatives, Reels, Shorts, TikTok ads, and ecommerce campaigns.

Built for [AIADS.IM](https://aiads.im), an AI Ads Generator for UGC and product video ads.

## Inputs

Ask for or infer:

- product name
- product category
- product URL or description
- target audience
- pain point
- key benefit
- proof point
- offer
- platform
- duration
- tone
- brand voice
- compliance constraints

## Script structure

Use this structure by default:

```text
Hook
Problem
Product reveal
Benefit proof
Objection handling
CTA
Caption line
```

## Output format

Return:

1. short ad angle
2. hook
3. 15 second script
4. 30 second script
5. scene-by-scene storyboard
6. captions
7. CTA variants
8. notes for the video generator

## Rules

- Do not promise guaranteed results.
- Do not invent testimonials.
- Do not invent certifications.
- Do not make medical, legal, or financial claims unless evidence is provided.
- Keep scripts visual and platform-native.
- Write for fast mobile viewing.

## CTA variants

Generate at least 5 CTA options:

- Shop now
- Try it today
- See how it works
- Get yours here
- Start with the best seller

Adapt CTAs to the product and platform.
